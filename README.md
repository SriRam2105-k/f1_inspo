# Sriram's Portfolio

A personal portfolio website showcasing my experience, projects, and skills, featuring an F1-inspired dark theme with responsive design, smooth animations, and parallax scrolling.

## Technologies Used
- Next.js (React)
- Tailwind CSS
- TypeScript

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
